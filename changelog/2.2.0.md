# Restricted Names 
You can no longer assign, nodes, scalars, directives and enums with the following name; 

_case insensitive_

```javascript
const RESTRICTED_NAMES = {
    query: 1,
    mutation: 1,
    subscription: 1,
    fields: 1,
    scalar: 1,
    enum: 1,
    union: 1,
    type: 1,
    interface: 1,
    int: 1,
    float: 1,
    string: 1,
    boolean: 1,
    id: 1,
    input: 1,
    directive: 1
};
```
