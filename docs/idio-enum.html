IdioEnum

const { IdioEnum } = require("idio-graphql");

Intro

You can use IdioEnum to modularize an EnumTypeDefinition, together with its resolver. You can specify enums 'top-level' at combineNodes or at an GraphQLNode level.

Example

const StatusEnum = new IdioEnum({
    name: "StatusEnum",
    typeDefs: `
    enum StatusEnum {
        ONLINE
        OFFLINE
        INACTIVE
    }
    `,
    resolver: {
        ONLINE: "online",
        OFFLINE: "offline",
        INACTIVE: "inactive"
    }
});
